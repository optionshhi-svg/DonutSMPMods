Installation

Follow these four simple steps to get started.

1

Install Fabric

Download the Fabric Loader from fabricmc.net.

2

Fabric API

Download the Fabric API .jar and drop it into your mods folder.

3

Drop the .jar

Place your downloaded tool .jar into the .minecraft/mods folder.

4

Launch & Play

Start Minecraft with Fabric. Press Right Shift to open the menu.
